### HTTP API 설계 예시

    1. HTTP API - 컬렉션

        POST 기반 등록
        



    2. HTTP API - 스토어

        PUT 기반 등록



    3. HTML FORM 사용

        GET, POST만 지원


### URI 설계 개념

    - 문서 : 단일 개념

            /members/100
            /files/star.jpg


    - 컬렉션 : 서버가 관리하는 리소스 디렉터리

            /members


    - 스토어 : 클라이언트가 관리하는 자원 저장소

            /file


    - 컨트롤러, 컨트롤러URI : 문서, 컬렉션, 스토어로 해결하기 어려운 추가 프로세스 실행

            /members/{id}/delete
